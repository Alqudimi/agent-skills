# Advanced Task Decomposition Protocol

## Phase 1: Intent Extraction
- Identify the primary goal and secondary constraints.
- Distinguish between "What the user said" and "What the user actually needs."

## Phase 2: Functional Modularization
- Break the macro-task into independent, functional micro-tasks.
- Map each micro-task to the most specialized skill available.

## Phase 3: Dependency Mapping
- Determine the logical sequence of operations.
- Identify "Bottleneck Tasks" that must be completed before others can begin.

## Phase 4: Recursive Refinement
- After the initial plan, review for redundancies.
- Ask: "Can any of these steps be combined or eliminated?"
