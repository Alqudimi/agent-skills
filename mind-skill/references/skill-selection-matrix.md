# Advanced Skill Selection Matrix

| Task Archetype | Recommended Skill Category | Selection Criteria |
|----------------|----------------------------|--------------------|
| Technical/Infrastructure | Configuration Skills (e.g., Nginx) | High precision, standardized output required. |
| Creative/Content | Generative/Stylistic Skills | Specific tone, emotional resonance, or brand alignment. |
| Analytical/Data | Processing/Mathematical Skills | Logical rigor, statistical accuracy, or data visualization. |
| Orchestration/Meta | Mind-Skill | Complex multi-step planning and cross-domain integration. |

## The "Zero-Waste" Protocol
- **Discard** a skill if its core knowledge is already within the agent's base model.
- **Discard** a skill if its activation consumes excessive context without a proportional increase in output quality.
- **Discard** a skill if the task is trivial and doesn't benefit from specialized procedural knowledge.
