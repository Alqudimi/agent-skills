# Agent-Skills: الإطار الإدراكي المتكامل لوكلاء الذكاء الاصطناعي

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com/Alqudimi/agent-skills)
[![Framework: OpenCode](https://img.shields.io/badge/Framework-OpenCode-green.svg)](https://opencode.ai)

**Agent-Skills** هو مستودع برمجيات عالي الأداء ومجزأ، صُمم لتحويل وكلاء الذكاء الاصطناعي التقليديين إلى أنظمة ذاتية التشغيل متخصصة وعالية الموثوقية. توفر هذه المجموعة "نظام التشغيل الإدراكي" للوكلاء العاملين ضمن بيئة **OpenCode**، مما يتيح تنسيقاً متقدماً، وتفكيراً استراتيجياً، وتنفيذاً بمستوى احترافي عبر مختلف المجالات التقنية.

---

## 🏗️ نظرة عامة على البنية الهيكلية

تم تصميم المستودع كإطار عمل استخباراتي متعدد الطبقات، حيث تعمل كل "مهارة" (Skill) كواحدة متخصصة يمكن تحميلها وتنسيقها وتوسيع نطاقها ديناميكياً.

| الطبقة | الوحدة | الوظيفة الأساسية |
| :--- | :--- | :--- |
| **الإدراكية** | `mind-skill` | التنسيق الاستراتيجي، اختيار الأدوات بذكاء (Zero-Waste)، وتفكيك المهام. |
| **الهيكلية** | `subagent-creator` | أنماط تنسيق الوكلاء المتعددين (هرمي، متوازي، تسلسلي). |
| **التشغيلية** | `make-me-better` | التميز الهندسي، إدارة المخاطر، وتحليل الأسباب الجذرية (RCA). |
| **المتخصصة** | `PDFMathTranslate` | ترجمة المستندات بدقة عالية مع الحفاظ على صيغ LaTeX والتنسيق. |
| **التكامل** | `opencode-agent-creator` | بروتوكولات تكوين ونشر وكلاء OpenCode الأصليين. |

## 🧠 تعمق في المهارات الأساسية

### 1. مهارة العقل (Mind-Skill): المنسق الإدراكي
الأساس الفكري للإطار العملي. تمنع "تشتت الأدوات" من خلال تطبيق التفكير من المبادئ الأولى على كل طلب.
- **الاقتصاد الاستراتيجي:** استخدام الحد الأدنى الضروري من الأدوات لتحقيق الهدف.
- **تفكيك المهام:** تقسيم الأهداف الكبرى إلى مهام دقيقة قابلة للتحقق باستخدام بروتوكول المراحل الأربع.
- **أطر التفكير:** دمج تفكير الأنظمة و"نصل أوكام" (Occam’s Razor) في حلقة تفكير الوكيل.

### 2. منشئ الوكلاء الفرعيين (Subagent-Creator): تنسيق الوكلاء المتعددين
تمكن من الانتقال من الوكلاء الأحاديين إلى "فرق الوكلاء" عالية الكفاءة.
- **أنماط التنسيق:** دعم أنماط **التوازي (Parallel)**، **الأنابيب التسلسلية (Sequential)**، و**الأشجار الهرمية (Hierarchical)**.
- **توجيه النماذج:** تعيين المهام للنماذج بناءً على تعقيدها (مثلاً: Haiku للاستكشاف، وOpus للتخطيط).
- **تحسين السياق:** تطبيق "وضع التفرع" (Fork Mode) لمشاركة ذاكرة التخزين المؤقت وتقليل استهلاك الرموز (Tokens) بنسبة تصل إلى 90%.

### 3. اجعلني أفضل (Make-Me-Better): التميز الهندسي
مجموعة صارمة من 53 توجيهاً أساسياً تحول الوكيل إلى مهندس معماري للأنظمة المحترفة.
- **يقين المعرفة:** يمنع اتخاذ إجراءات بناءً على بيانات غير كافية؛ ويفرض الاستيضاح النشط.
- **إدارة النزاهة:** يضمن استقرار النظام بالكامل أثناء عمليات إعادة الهيكلة (Refactoring) الحرجة.
- **الاستباقية:** التنبؤ بنقاط الفشل المستقبلية واقتراح تحسينات معمارية.

### 4. مترجم الرياضيات (PDFMathTranslate): ترجمة المستندات الاحترافية
أدوات متخصصة لمعالجة المستندات الأكاديمية والتقنية.
- **الحفاظ على التنسيق:** الحفاظ على معادلات LaTeX المعقدة، الجداول، والتنسيقات متعددة الأعمدة.
- **مخرجات مزدوجة:** إنشاء ملفات PDF ثنائية اللغة (جنباً إلى جنب) للمرجعية أو لغة واحدة للتسليم النهائي.
- **تكامل OCR:** دعم مدمج للمستندات الممسوحة ضوئياً عبر Tesseract.

## 🚀 البدء السريع

### التثبيت
قم باستنساخ المستودع في دليل تكوين OpenCode الخاص بك أو في جذر المشروع:

```bash
git clone https://github.com/Alqudimi/agent-skills.git
```

### نشر وكيل
لإنشاء وكيل متخصص (مثل مدقق أمني)، أنشئ ملف `.md` في المسار `.opencode/agents/`:

```markdown
---
description: مراجعة الكود بحثاً عن الثغرات الأمنية
mode: subagent
model: anthropic/claude-sonnet-4-20250514
permission:
  edit: deny
  bash: deny
  read: allow
---
# مدقق أمني
أنت مراجع يركز على الأمن. قم بتحليل الكود بحثاً عن ثغرات SQLi و XSS وعيوب المصادقة.
```

## 🛡️ بروتوكولات الأمن والموثوقية

تطبق Agent-Skills **نموذج أذونات "عدم الثقة" (Zero-Trust)**:
- **الحد الأدنى من الصلاحيات:** جميع الأدوات معطلة (`deny`) افتراضياً.
- **بوابات الموافقة:** العمليات الحساسة (مثل `rm` أو `sudo` أو `push`) تتطلب تأكيداً صريحاً من المستخدم.
- **قواطع التيار (Circuit Breakers):** اكتشاف حلقات الوكيل اللانهائية أو الإخفاقات المتكررة تلقائياً لمنع هدر الرموز.
